About this project

Physics Engine

A web interface for a C++ stellar physics engine. Computes real astrophysical quantities using Stefan–Boltzmann law, Wien's displacement law, and Schwarzschild general relativistic effects.

The gravitational redshift factor √(1 − rₛ/R) determines photon escape conditions. Stellar evolution uses mass-luminosity relations to estimate lifetimes across three recursive stages.

Updated v3 adds Planck-law emission spectra, a live Hertzsprung–Russell diagram, Morgan–Keenan spectral classification, bolometric magnitudes, and physically accurate blackbody color — computed with CODATA 2018 / IAU 2015 nominal constants.

// Stefan-Boltzmann + Wien's law void analyzeStar(Star &s){ s.L=4*pi*s.R*s.R*sigma*pow(s.T,4); s.lambda=b/s.T; s.F=s.L/(4*pi*s.d*s.d); s.schw_R=(2*G*(s.i_mass*solar_mass))/(c*c); // Schwarzschild redshift (GR) if(s.R <= s.schw_R) s.status="Black hole - no escape"; else s.redshift_F=sqrt(1.0-(s.schw_R/s.R)); } // Fading Red Dwarf branch (v2) if(s.i_mass < 0.5){ s.evo_status="Fading Red Dwarf"; evolve(s, age, 2); return; }
⚡
Stefan–Boltzmann
L = 4πR²σT⁴ — luminosity from surface area and temperature with full double precision.
🌀
Gravitational Redshift
Schwarzschild factor √(1 − rₛ/R) models photon energy loss escaping a gravitational well.
⏳
Stellar Evolution
Recursive 3-stage timeline with Fading Red Dwarf, Red Giant, and Supergiant branches.
🔬
Planck Spectra
Full B_λ(λ,T) spectral radiance curves with Wien peak, rendered over the true-color visible band.
📡
Flux & Pressure
F = L/(4πd²) and P = F/c with detection threshold classification.
🔀
H–R Diagram & Export
Every star plotted live on a Hertzsprung–Russell diagram, with CSV export and a persistent registry.
